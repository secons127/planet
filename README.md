## 🎯 개요 및 목적

이 프로젝트는 **식물을 키우고 싶지만 두려움이 있는 초보자들**을 위해 기획되었습니다.

### 핵심 가치
- 🌱 **실전 연습**: 가상 환경에서 식물 관리법을 배우고 자신감을 얻을 수 있습니다
- 🤖 **AI 상담**: 식물 전문가 AI와 대화하며 문제를 해결합니다
- 📝 **성장 기록**: 실제로 키우는 식물의 일지를 작성하고 관리합니다
- 🎮 **게임화**: 재미있는 게임 요소로 식물 관리를 배웁니다

---

## ✨ 주요 기능

### 1. 🌱 식물 키우기 (게임 모드)
가상 식물을 키우며 식물 관리의 기초를 배웁니다.

#### 성장 시스템
- **4단계 성장**: 씨앗(Lv 1-3) → 새싹(Lv 4-6) → 성장기(Lv 7-9) → 성체(Lv 10+)
- **경험치 획득**: 물주기, 비료, 대화 등으로 경험치를 얻어 레벨업
- **점진적 난이도**: 성장할수록 관리해야 할 요소가 증가

#### 관리 요소
- 💧 **물주기**: 수분 게이지 관리
- 💊 **영양제**: 상태이상 치료 또는 성장 촉진 (2배 경험치)
- 🌬️ **환기하기**: 통풍 관리 (실내 전용)
- ✂️ **가지치기**: 위생 관리 (성장기부터 해금)
- ✨ **잎 닦기**: 광합성 효율 증가 (새싹부터 해금)

#### 환경 시스템
**배경 선택**:
- 🏠 거실 (기본)
- 🪟 베란다
- 🏢 사무실
- 🌲 숲 (야외)
- 🏜️ 사막 (야외)

**야외 환경 특징**:
- 날씨 시스템 (☀️ 맑음 / 🌧️ 비 / ☁️ 흐림)
- 자동 환기 (통풍 100% 유지)
- 화분 없이 땅에서 성장
- 환경별 특성:
  - **숲**: 위생 감소 50% 느림 (깨끗한 공기)
  - **사막**: 수분 감소 2배 빠름 (건조함)

#### 날씨 영향
- **맑음**: 수분 증발 가속
- **비**: 자동 물 공급 + 잎 세척, 과습 위험 증가
- **흐림**: 수분 증발 감소

#### 상태 관리
- **토양 상태**: 건조 / 적정 / 과습
- **건강 상태**: 건강 / 해충 / 뿌리썩음 / 시듦
- **검사 기능**: 현재 상태를 상세히 확인

### 2. 📝 식물 일지
실제로 키우는 식물의 성장을 기록하고 관리합니다.

#### 내 식물 등록
- 식물 종류 선택 (사전의 모든 식물)
- 애칭 설정 (예: "거실 몬스테라")
- 위치 기록 (예: "거실 창가")
- 입양일 설정
- 함께한 일수 자동 계산

#### 일지 작성
**활동 유형**:
- 💧 물주기
- 💊 비료
- ✂️ 가지치기
- 🪴 분갈이
- 📝 일반 메모

**기록 항목**:
- 활동 날짜 (자동)
- 메모 내용
- 식물 상태 (😊 아주 좋음 / 🙂 좋음 / 😐 보통 / 😟 걱정됨)

#### 피드 조회
- 타임라인 형태로 모든 일지 확인
- 최신순 자동 정렬
- 날짜별 표시 (오늘/어제/N일 전)
- 식물별 필터링 가능

#### 데이터 저장
- LocalStorage 자동 저장
- 브라우저 종료 후에도 데이터 유지
- 별도 로그인 불필요

---

### 3. 📚 식물 사전
다양한 식물의 정보를 확인하고 나에게 맞는 식물을 찾습니다.

#### 난이도별 분류
- 🟢 **초보용 (Easy)**: 스투키, 산세베리아, ZZ플랜트
- 🟡 **도전용 (Medium)**: 몬스테라, 고무나무
- 🔴 **프로용 (Hard)**: 칼라데아, 떡갈고무나무

#### 상세 정보 (식물 여권)
- 📸 식물 사진
- 🌍 원산지
- 🏷️ 학명 및 과명
- 💧 물주기 빈도
- ☀️ 빛 요구량
- 🌡️ 적정 온도
- 📋 상세 관리 가이드

---

### 4. 💬 AI 상담소
식물 전문가 AI와 대화하며 궁금증을 해결합니다.

#### 기능
- **실시간 채팅**: Gemini AI 기반 대화
- **전문 상담**: 식물 관리, 병해충, 환경 등 모든 질문 답변
- **페르소나**: 친근한 식물 전문가 컨셉
- **대화 기록**: 채팅 히스토리 유지

#### 사용 예시
- "몬스테라 잎이 노랗게 변했어요"
- "겨울철 물주기 주기는 어떻게 되나요?"
- "베란다에서 키우기 좋은 식물 추천해주세요"

## 🛠️ 기술 스택

### Frontend
- **Framework**: React 18 + TypeScript
- **Build Tool**: Vite
- **Styling**: TailwindCSS
- **State Management**: Zustand (persist middleware)
- **Icons**: Lucide React
- **Routing**: React State 기반 SPA

### Backend
- **Framework**: FastAPI (Python)
- **AI**: Google Gemini API (`gemini-1.5-flash`)
- **Environment**: Python 3.10+

### 개발 환경
- **Runtime**: Node.js 18+
- **Package Manager**: npm
- **Version Control**: Git

## 🚀 설치 및 실행

### 1. 저장소 클론
```bash
git clone https://github.com/secons127/planet.git
cd planet
```

### 2. Frontend 설정
```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev
```
- 브라우저에서 `http://localhost:5173` 접속

### 3. Backend 설정 (AI 상담소 사용 시)
```bash
cd backend

# 가상환경 생성 및 활성화 (Windows)
python -m venv .venv
.venv\Scripts\activate

# 의존성 설치
pip install -r requirements.txt

# 환경 변수 설정
# backend/.env 파일 생성 후 API 키 입력
echo GEMINI_API_KEY=your_api_key_here > .env

# 서버 실행
python -m uvicorn main:app --port 8000
```

### 4. 프로덕션 빌드
```bash
npm run build
npm run preview
```

## 📱 사용 방법

### 첫 시작
1. 홈 화면에서 원하는 기능 선택
2. **식물 키우기**: 가상 식물로 연습
3. **식물 일지**: 실제 식물 등록 및 기록
4. **식물 사전**: 정보 탐색
5. **AI 상담소**: 궁금한 점 질문

### 식물 키우기 팁
1. **초반**: 물만 주면서 레벨 3까지 성장
2. **새싹 (Lv 4)**: 잎 닦기 해금, 위생 관리 시작
3. **성장기 (Lv 7)**: 가지치기 해금, 전체 관리 필요
4. **야외 환경**: 날씨 확인하며 과습 주의

### 일지 작성 팁
1. **규칙적 기록**: 물 준 날짜를 꼭 기록
2. **상태 관찰**: 잎 색깔, 새 잎 등 변화 메모
3. **사진 활용**: 성장 과정 시각적 기록 (향후 지원 예정)

## 🎮 게임 메커니즘

### 경험치 획득
- 물주기: +5 Exp
- 영양제 (건강): +60 Exp (2배)
- 영양제 (치료): 상태이상 제거
- 대화하기: +5 Exp
- 가지치기: +15 Exp
- 환기하기: +5 Exp

### 레벨 시스템
- 각 레벨마다 필요 경험치 1.5배 증가
- 3레벨마다 성장 단계 변화
- 성장 단계별 새로운 기능 해금

### 생존 요소
- 수분: 시간당 자동 감소 (환경별 차이)
- 위생: 먼지 누적 (환경별 차이)
- 통풍: 실내에서 자동 감소
- 건강: 조건 충족 시 상태이상 발생
